# 💜 ENHYPEN Bias Experience

Welcome to the **ENHYPEN Bias Experience**, an interactive fan website dedicated to the seven members of ENHYPEN.

This project allows fans to choose their favorite member (bias), discover their special talents, and learn interesting facts about them through a fun and interactive experience.

---

## ✨ Features

- 🌌 Beautiful welcome page with ENHYPEN OT7
- 💜 Interactive bias selection
- ⭐ Personalized talent page for each member
- 🎂 Member profile with:
  - Birthday
  - Position
  - Nationality
  - Zodiac Sign
  - Fun Fact
- 🎨 Modern purple aesthetic design
- 📱 Mobile responsive layout
- ✨ Smooth page transitions and animations

## 🚀 How to Run

1. Download or clone this repository.
2. Make sure all images are inside the **images** folder.
3. (Optional) Add your own background music to the **music** folder.
4. Open **index.html** in your web browser.
